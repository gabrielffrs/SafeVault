# SafeVault-Security 🔒

Sistema de monitoramento de sensores e gestão de protocolos de segurança.

## 🛠️ Funcionalidades
- Autenticação de operador via ID funcional com barramento de acesso.
- Monitoramento de telemetria ambiental (Energia, Oxigênio e Temperatura).
- Disparo de protocolos de emergência automatizados via lógica de decisão (`switch-case`).

## 💻 Tecnologias
- Java 25.
- Scanner para processamento de entradas em tempo real.
- Estruturas de decisão aninhadas para controle de escopo.

## 📝 Como rodar
1. Clone o repositório.
2. Compile o arquivo `SafeVault.java`.
3. Execute e siga as instruções no console.
